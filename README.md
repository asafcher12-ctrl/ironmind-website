# IronMind - אתר נחיתה

אתר נחיתה מקצועי לעסק מסגרות - שערים, פרגולות ומעקות.

## קבצים
- `index.html` - האתר המוכן לפרסום (HTML מלא עם תמונות)
- `LandingPage.jsx` - קומפוננט React לפיתוח עתידי
- `ironmind-logo.png` - לוגו העסק

## פרסום ב-GitHub Pages
1. העלה את הקבצים ל-GitHub
2. Settings → Pages → Deploy from branch → main
3. האתר יהיה חי בכתובת: `https://USERNAME.github.io/ironmind-website/`

## WhatsApp
עדכן את מספר הטלפון בקובץ `index.html` - חפש: `WHATSAPP_NUMBER`
